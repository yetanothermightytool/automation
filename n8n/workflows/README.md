# Sample Workflows

This directory contains some sample workflows that can be imported into n8n. Some adjustments might be necessary, e.g. assigning the correct credentials.


## Create Malware Event.json
Creates a new malware event. (Veeam Incident API). Adjust the parameters in the Set Node.
## Get All Authorization Events.json

## Get All Malware Events.json
Gets malware events created on the backup server.

## Get All Triggered Alarms - VONE.json
This workflow shows all security-related alarms with Warning or Error status.

## Get Restore Points.json
Gets the restore points created on the backup server. Adjust the parameters in the HTTP Query Node.

## Get Security & Compliance Analyzer Last Run.json
Gets the last session of Security & Compliance Analyzer.

## Get Security & Compliance Analyzer Results.json
Gets the best practices compliance results from the backup server.

## Start Quick Backup.json
Starts a quick backup for the specified VMware vSphere VM. Adjust the parameters in the Set Node.

## Start Security & Compliance Analyzer.json
Starts the Security & Compliance Analyzer on the backup server.
